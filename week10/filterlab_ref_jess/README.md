# Filter Lab

This is an optimized bloom filter implementation created by Jess during Fall 2024 
while taking 98-008: Intro to the Rust Programming Language. Jess later became an 
instructor for the course in Spring 2025.

This implementation focuses on reducing false positives via thoughtful hashing algorithms (the 99%) before SIMD optimization (the 1%).

Read the full writeup [here](https://jessicaruan.com/posts/filterlab).
